# Credit Card Fraud Analysis Dashboard | Power BI

## Project Overview
Developed an interactive Credit Card Fraud Analysis Dashboard in Power BI to monitor fraudulent transactions, analyze fraud patterns, and identify high-risk transaction categories. The project utilizes transaction-level data from CSV files and demonstrates end-to-end data modeling, DAX calculations, KPI reporting, and interactive dashboard development.

## Key Features
* Designed a Star Schema data model using Fact and Dimension tables.
* Created calculated columns and DAX measures for fraud analytics.
* Developed KPI cards to track:
  * Fraudulent Transactions
  * Fraudulent Transactions Amount
  * Fraud Type
  * Fraud Rate (%)
  * Critical Risk Transaction
* Implemented interactive slicers for dynamic filtering and drill-down analysis.
* Analyzed fraud distribution across card types, transaction categories, banks, merchants, and locations.
* Identified transaction patterns and high-risk fraud categories.

## Tools & Technologies
* Power BI
* DAX
* Data Modeling
* CSV Data Source
* Power Query

## Business Value
The dashboard enables stakeholders to monitor fraud trends, evaluate transaction risk, and support data-driven decision-making for fraud prevention and investigation.
